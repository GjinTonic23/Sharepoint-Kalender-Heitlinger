# Kalender Web App Backend

Starte das Backend mit:

    cd backend
    npm install
    npm run dev

Das Backend lädt die Kalenderdaten aus der CSV-Datei und stellt sie per REST-API bereit.

# Kalender Web App Frontend

Starte das Frontend mit:

    cd frontend
    npm install
    npm start

Das Frontend zeigt die Kalenderdaten an und wird weiter ausgebaut.

# Hinweise
- Die CSV-Datei muss im Projektverzeichnis liegen (../VA_2026_SHAREPOINT_OPTIMIERT_v2.csv)
- Weitere Features werden Schritt für Schritt ergänzt.
