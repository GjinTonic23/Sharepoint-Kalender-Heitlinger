# Shared Calendar Web App

Dieses Projekt ist eine moderne Webanwendung für einen gemeinsam nutzbaren Kalender. Sie besteht aus einem React-Frontend und einem Express-Backend. Die Kalenderdaten werden aus einer CSV-Datei geladen und können von mehreren Nutzern gleichzeitig bearbeitet werden.

## Features
- Kalenderansicht mit mehreren Veranstaltungen pro Tag
- Letzter Bearbeiter pro Tag sichtbar
- Zeilen mit Datum in der Spalte "Option" werden blau markiert
- Spalten "Kreuzberghof" und "AGB" entfernt, "Angebot" in "Zugesagt" umbenannt
- Veranstaltungen können hinzugefügt, bearbeitet und gelöscht werden
- Multi-User-Zugriff

## Projektstruktur
- `backend/` – Express-Server, CSV-Import, REST-API
- `frontend/` – React-App für die Kalenderoberfläche
